## 🛠️ Installation du Backend

### Prérequis
- PHP 8.5+
- [Composer](https://getcomposer.org/download/) installé globalement

### Installation
1. Clonez le projet.
2. Accédez au dossier : `cd backend_admin`.
3. Copiez `.env.example` en `.env`, adaptez les valeurs
4. Installez les dépendances :
   ```bash
   composer install
